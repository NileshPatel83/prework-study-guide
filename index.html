HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The element is between starting and closing tag whereas tag is between angle brackets.
  • Body element is called parent element whereas main element is called child element.
  • Element can also contain attribute(s) that performs certain task.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS must be linked to HTML file to apply the style to HTML page.
  • CSS can be inline, internal or external. External CSS is preferred as easy to maintain and separation of concern
  • CSS has selector (can have multiple selector separated by semicolon) that decides where to apply the style. Attribute and value to apply to the selector.
  • There are times where you don't want to apply the same style to all elements. In this case, use "Class" attribute as selector to apply the style to specific elements that has this attribute in HTML element.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git commit -m "Description": Commints the current changes.
  • git pull remote branch: Pulls the latest chagnes from specified branch from remote repo to local repo.
  • git push remote branch: Pushes local changes to specified branch in remote repo.

JavaScript

  • JavaScript is a programming language that allows users to interact with websites.
  • JavaScript file is linked to element in HTML file.
  • JavaScript code is executed from top to bottom therefore order is important but can be interrupted using conditions.
  • Each line in JavaScript must end with semicolon except for conditions and loops
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Function must end with parentheses.
  • Function must be called to execute the codes inside the function.
